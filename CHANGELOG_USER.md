# 金快查 用户更新日志

## v1.2.0 (2026-05-16)

### 新增功能

- **场内份额追踪**：集成上交所/深交所官方 API，展示 LOF 基金场内份额及新增份额变化（单位：万份）
- **申购限额筛选**：新增下拉多选框，支持按"暂停申购"、"开放申购"、具体限额筛选基金，一键清空
- **定时任务系统**：每日 07:00 自动抓取 T-1 日份额数据，非交易日自动跳过
- **365天历史数据**：历史数据保留期从 21 天扩展到 365 天，支持更长周期的趋势分析
- **任务状态查询**：新增后台任务管理接口，可查看定时任务执行状态

### 体验优化

- **移动端卡片优化**：移动端显示"新增份额xxx万"，更直观展示资金流向
- **排行榜交互**：点击溢价/折价 TOP5 基金可直接弹出详情弹窗
- **KPI 网格居中**：基金详情弹窗 KPI 内容居中对齐，视觉更平衡
- **日期显示**：估算净值、场内份额、新增份额下方显示数据日期
- **单位标注**：表头明确标注"(万份)"单位，单元格纯数字显示
- **头部布局优化**：主题切换和刷新按钮靠右对齐，状态栏移至侧边栏
- **文档导航增强**：README.md 添加多处文档跳转链接，方便快速查找

### 数据与性能

- **数据源扩展**：新增交易所官方 API 作为份额数据源，所有数据源均为公开免费接口
- **熔断机制优化**：连续失败 3 次自动降级备用源，5 分钟后自动重试主源
- **图表缓存升级**：热门基金图表预渲染 Top5 溢价 + Top5 折价，每 5 分钟刷新
- **数据库优化**：新增 `daily_kline` 表存储 365 天日线数据，查询效率提升

### 代码结构

- **模块化重构**：数据源统一整合至 `backend/datasource/` 目录，采用策略模式
- **依赖管理优化**：`requirements.txt` 移至根目录，符合 Python 项目规范
- **环境变量完善**：新增 `.env.example` 模板文件，简化配置流程
- **Git 忽略规则**：完善 `.gitignore`，忽略虚拟环境、临时文件等

### 文档体系

- **README 全面升级**：反映 v1.2.0 最新状态，添加贡献者章节
- **PR 文档准备**：新增 `PULL_REQUEST_TEMPLATE.md`、`PR_DESCRIPTION.md`、`MERGE_GUIDE.md`
- **技术文档恢复**：恢复 `CHANGELOG.md`、`ENV_SETUP.md` 等技术文档
- **协作模式明确**：README 中添加 MistyBridge 和 danwangshi 贡献者信息

## v1.1.0 (2026-05-16)

### 新增功能

- **历史走势图**：基金详情页新增 365 天日线走势图，支持七日/一月/三月/六月/一年切换
- **溢价率曲线**：新增溢价率独立曲线图，红色为正溢价、绿色为折价，Y 轴颜色同步
- **套利模拟**：鼠标悬停走势图任意日期，自动计算该日作为套利起点的预计收益（溢价套利 T 日净值申购→T+2 卖出 / 折价套利 T 日买入→T+1 赎回，取收益更高者）
- **申购限额**：基金详情页显示申购限额，支持"不限额"标识，停售基金自动隐藏
- **曲线指标切换**：左侧下拉菜单切换"场内价格+场外净值"或"溢价率"图表
- **深色模式**：全局适配深色/浅色模式，滚动条、弹窗、图表坐标轴同步切换

### 体验优化

- 图表加载不再突然弹出，先显示空坐标系再填充数据
- 收益构成明细改为浮动弹窗，不挤占页面空间
- 套利提示改为软悬浮窗，悬停时出现在图表上方不遮盖曲线
- 基金详情弹窗改为上下对称居中，视觉更平衡
- 代码/名称旁标注"(点击可复制)"
- PC 端表格列宽优化，长名称不再换行，悬停显示完整名称
- 自定义滚动条样式，与页面主题一致

### 数据与性能

- 日线数据覆盖 396 只活跃 LOF 基金，近一年历史行情
- 实时净值自动区分盘中估算与盘后正式净值，溢价率计算更准确
- 热门基金图表预渲染缓存，打开即显无需等待
- 数据源智能切换，主源故障自动降级备用源，恢复后自动切回
