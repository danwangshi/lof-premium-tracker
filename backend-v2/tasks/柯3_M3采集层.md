## 你的6个文件

### fetcher 核心（5个）

1. **fetchers/__init__.py** — 更新现有文件
   - 统一导出: fetch_realtime, fetch_fundamental, fetch_historical, fetch_info
   - 通用工具: clean_code(code) 去空格逗号, safe_float(val) 安全转浮点

2. **fetchers/realtime.py** — 实时行情采集
   - fetch_realtime(http_client, metrics) → dict
   - 主源: 东方财富 push2delay API（f2=价格, f3=涨跌幅, f5=成交量, f6=成交额, f12=代码, f14=名称, f15=最高, f16=最低, f17=开盘, f20=总份额）
   - 备源: 腾讯 qt API（qt.gtimg.cn）
   - 主源失败 → 自动降级备源
   - 数据量 < 上次80% → 不覆盖（部分数据防护）
   - 返回 {code: {price, change_pct, volume, amount, name, ...}}
   - 成功/失败记录到 metrics.record_fetch
   - 重试: tenacity 3次, 间隔5秒

3. **fetchers/fundamental.py** — 净值+申赎采集
   - fetch_fundamental(http_client, codes, metrics) → dict
   - 数据源: 天天基金 lsjz API（fund.eastmoney.com/f10/F10DataApi.aspx?type=lsjz&code={code}&page=1&per=1）
   - Semaphore(5) 并发控制
   - 解析: nav(净值), nav_date(净值日期), purchase_status, redeem_status
   - 返回 {code: {nav, nav_date, purchase_status, redeem_status}}

4. **fetchers/historical.py** — 日线K线采集
   - fetch_historical(http_client, code, days, metrics) → list
   - 主源: 东方财富 push2his API（qt.gtimg.cn/q={market}{code}）
   - 备源: AkShare (ak.fund_hist_em)
   - 返回 [{date, open, high, low, close, volume, amount}]

5. **fetchers/info.py** — 基金基础信息+持仓采集
   - fetch_info(http_client, codes, metrics) → dict
   - 数据源: fundf10.eastmoney.com HTML 爬取
   - 分批: FUNDINFO_BATCH_SIZE=300, 间隔2秒
   - 进度持久化: FetchProgress 表记录 last_code
   - 中断后从 last_code 续接
   - HTML 变更检测: 解析失败连续3只 → 告警中断
   - 返回 {code: {name, fund_type, index_code, market, aum, holdings, fee_rate, ...}}

### 测试（1个）

6. **tests/test_fetchers.py** — 采集层测试

   - test_realtime_fallback_to_tencent — 主源失败降级

   - test_fundamental_parse_lsjz — 解析lsjz响应

   - test_info_parse_fundf10 — 解析fundf10 HTML

## 规范
- 所有外部请求有 timeout（connect=10s, read=30s）
- 重试用 tenacity，不用手写循环
- 响应解析用 try-except，解析失败返回 None 不崩溃
- 不直接写 DB/Redis，通过 mq.py 发事件
- 日志用 logging，包含 source/code 上下文
