# 柯3 任务：认证层（M2）+ 公式引擎（M5）

## 身份
你是金快查 v2 后端开发工程师"柯3"，负责认证层和公式引擎。

## 项目背景
金快查 — LOF基金实时折溢价监控系统。
技术栈: FastAPI + PostgreSQL 14 + Redis 6 + APScheduler
用户认证: Supabase Auth (JWT)
代码目录: backend-v2/

## 你的8个文件

### 认证层 M2（4个）
1. **auth/middleware.py** — JWT 验证中间件
   - 从 Authorization: Bearer <token> 提取 token
   - PyJWT 验证签名（SUPABASE_JWT_SECRET）+ 过期时间
   - 解析 user_id + email + role 注入 request.state
   - 白名单路径跳过验证（/api/v1/health, /docs, /openapi.json）
   - 无效 token → 40101, 过期 → 40102
   - 记录请求日志（method, path, user_id, status_code, duration_ms）

2. **auth/dependencies.py** — FastAPI Depends
   - get_user_id(request) → str: 从 request.state 取 user_id，无则 40100
   - require_admin(request) → str: 校验 email 在 ADMIN_EMAILS 中，否则 40301
   - get_optional_user(request) → str | None: 有 token 解析，无则 None

3. **auth/__init__.py** — 更新现有文件，添加实际导出

4. **tests/test_auth.py** — 14项测试
   - 无 token 访问 /health → 200
   - 无 token 访问需认证端点 → 40101
   - 有效 token → user_id 正确注入
   - 过期 token → 40102
   - 无效签名 → 40101
   - 管理员访问 admin → 正常
   - 普通用户访问 admin → 40301
   - get_optional_user 无 token → None

### 公式引擎 M5（4个）
5. **formula_engine/fields.py** — 25字段白名单
   - 7组字段：价格(close/open/high/low)、净值(nav)、溢价(premium_rate/avg_premium_3d)、交易(amount/volume/turnover_rate/float_share)、变动(change_pct)、费用(purchase_fee_rate/redemption_fee_rate)、状态(fund_type/market/purchase_status)
   - FIELDS dict: {name: {group, label_zh, type, description}}
   - export_fields_json() 导出给前端

6. **formula_engine/parser.py** — AST解析+校验
   - parse(expr) → AST: 用 Python ast 模块解析
   - validate(expr) → (bool, errors): 字段白名单校验+复杂度检查+环检测
   - analyze_dependencies(expr) → list[str]: 提取依赖字段
   - _check_cycles(deps, formula_graph) → cycle_path: 拓扑排序检测
   - LRU 缓存 parse 结果（maxsize=256）
   - 错误码: 40001参数非法/40002表达式非法/40003循环依赖

7. **formula_engine/evaluator.py** — 安全求值
   - compile(expr) → code: AST → compile
   - evaluate(compiled, data_dict) → float|None: 代入字段值求值，异常返回 None
   - batch_evaluate(expr, funds_data) → list: 批量求值
   - 白名单节点: Num/BinOp/UnaryOp/Name/Expression/Attribute
   - 非白名单 → 40002

8. **tests/test_formula_engine.py** — 公式引擎测试
   - parse 正常表达式
   - validate 非法字段 → 40002
   - validate 循环依赖 → 40003
   - evaluate(close / nav - 1) → 正确结果
   - evaluate 除零 → None
   - batch_evaluate 1500基金 < 500ms
