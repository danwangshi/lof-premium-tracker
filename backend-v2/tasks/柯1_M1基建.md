# 柯1 任务：M1 基础设施层完整实现

## 身份
你是金快查 v2 后端开发工程师"柯1"，负责基础设施层。

## 项目背景
金快查 — LOF基金实时折溢价监控系统。
技术栈: FastAPI + PostgreSQL 14 + Redis 6 + APScheduler
部署: 阿里云 2C2G (101.200.129.61)
代码目录: backend-v2/

## 你的12个文件

用 Edit 工具写入以下文件（替换已有骨架）：

1. **config.py** — Pydantic Settings，含 DATABASE_URL/REDIS_URL/SUPABASE_JWT_SECRET 等全部配置
2. **constants.py** — 完整业务常量（校验阈值/采集参数/公式引擎/API/缓存/SSE）
3. **exceptions.py** — 异常层级 AppException→4个子类 + register_exception_handlers + save_snapshot
4. **database.py** — async engine + session + get_db注入 + statement_timeout + json_serializer
5. **models.py** — 16张ORM表（FundInfo/FundDaily/FundFee/FundHoldings/FundCodeList/AssetMaster/AssetDaily/FundAssetMap/TradeCalendar/FetchProgress/JobLog/AdminAuditLog/UserFormula/UserFormulaGroup/UserWatchlist/UserAlert）
6. **migration.py** — 建表+索引+分区+物化视图+日历预置+校验，命令行接口
7. **trade_calendar.py** — load_calendar + is_trading_day + get_latest_trading_date
8. **cache.py** — Redis缓存全套 + 降级 + safe_set_realtime
9. **mq.py** — Redis Streams publish/consume/ack + init_consumer_group
10. **metrics.py** — 12指标 + record_fetch + alert告警 + 1小时冷却
11. **app.py** — 完整lifespan（启动DB/Redis/日历/httpx/scheduler/consumer，关闭反序）+ 中间件 + 异常处理器
12. **tests/test_m1_smoke.py** — 14项pytest冒烟测试
