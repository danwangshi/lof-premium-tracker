## 二、数据流

```
Stream 事件
  → pipeline.dispatch(event)
    → normalize（字段映射 + 类型统一 + code.zfill(6)）
    → validator（校验 + 去重 + 涨跌停标记 + 异常检测）
    → 分支处理：
        realtime → safe_set_realtime（部分数据防护）+ set rt:close:{date}
                   + 后台 asyncio.create_task(cross_validate_akshare)
        nav      → Redis nav:all (TTL 300s)
        kline    → Redis kline:fund:{date} / kline:asset:{date} (TTL 次日凌晨)
        info     → DB fund_info/fund_fee/fund_holdings (分批事务)
                   + Redis info:{code} + fee:{code}
        daily_save → 从 Redis 读取 → calculator → saver → 刷新物化视图
                     → 非 QDII 先入库，QDII 延迟重试
```

### 3.2 类型安全转换函数

```python
def to_optional_float(value) -> float | None:
    """
    安全转浮点，处理所有"空值"变体。
    注意：0 不是空值，成交量=0 表示停牌，是有效数据。
    """
    if value is None:
        return None
    if isinstance(value, str):
        value = value.strip()
        if value in ("", "-", "--", "暂无", "暂无数据", "N/A", "nan", "NaN"):
            return None
    try:
        result = float(value)
        if math.isnan(result) or math.isinf(result):
            return None
        return result
    except (ValueError, TypeError):
        return None

def to_optional_int(value) -> int | None:
    """安全转整数（成交量等）"""
    f = to_optional_float(value)
    return int(f) if f is not None else None

def clean_code(code) -> str | None:
    """基金代码统一清洗"""
    if code is None:
        return None
    code = str(code).strip().replace(" ", "").replace(",", "")
    code = code.lstrip("0") or "0"  # 全零保留一个
    code = code.zfill(6)
    if not code.isdigit() or len(code) != 6:
        return None
    return code
```

### 3.3 关键注意事项

**0 值处理：**
- `to_optional_float("0")` 返回 `0.0`，不返回 None
- 成交量=0（停牌）、涨跌幅=0（平盘）、换手率=0（无交易）都是有效值
- 只有 `None`、`""`、`"-"`、`NaN` 才返回 None

**Decimal 处理：**
- DB 读出的 NUMERIC 字段是 Python Decimal，不是 float
- normalize 入口统一转 float：`float(value) if isinstance(value, Decimal) else value`
- 避免 `Decimal("0.1") + float(0.2)` 混用产生精度问题

**名称处理：**
- 以 push2 返回的 name 为准（最权威，更新最快）
- 其他来源（fundf10、lsjz）的名称不入库，避免不一致

**source 参数：**
- normalize_realtime 需要区分 push2 和腾讯的字段映射
- push2: `f2`=最新价, `f3`=涨跌幅, `f5`=成交量
- 腾讯: 字段格式不同，需要独立映射表

```python
def mark_limit_status(record: dict) -> dict:
    price = record.get("realtime_price") or record.get("close")
    limit_up = record.get("limit_up")
    limit_down = record.get("limit_down")

    if price and limit_up and abs(price - limit_up) < 0.001:
        record["risk_warning"] = "涨停标的"
    elif price and limit_down and abs(price - limit_down) < 0.001:
        record["risk_warning"] = "跌停标的"

    return record
```

### 4.4 异常检测（detect_anomalies）

**历史数据来源：** 从 Redis 或 DB 批量获取近 20 日数据，不是逐只查询。

```python
async def detect_anomalies(records: list[dict], history_map: dict) -> list[dict]:
    """
    records: 当日数据
    history_map: {code: [近20日记录]}，批量预加载，不在函数内查询
    """
    for record in records:
        code = record.get("code")
        history = history_map.get(code, [])
        if len(history) < 5:
            continue  # 历史数据不足，跳过检测

        # 收盘价偏离 20 日均值 > 30%
        close = record.get("close")
        if close and history:
            avg_close = sum(h["close"] for h in history if h.get("close")) / len(history)
            if avg_close > 0 and abs(close - avg_close) / avg_close * 100 > PRICE_DEVIATION_WARN:
                record["risk_warning"] = "price_anomaly"

        # 净值突变 > 80%
        nav = record.get("nav")
        if nav and history:
            last_nav = history[-1].get("nav")
            if last_nav and last_nav > 0:
                nav_change = abs(nav - last_nav) / last_nav * 100
                if nav_change > NAV_JUMP_CRITICAL:
                    record["risk_warning"] = "nav_jump"
                    await alert("净值突变", f"{code}: {last_nav} -> {nav_change:.1f}%", "P1")

    return records
```

**性能优化：** history_map 在 pipeline.py 中批量预加载（一次 DB 查询获取所有基金的近 20 日数据），不在 detect_anomalies 内逐只查询。

### 5.2 换手率公式详解

```python
def calc_turnover_rate(volume: float, float_share: float) -> float | None:
    """
    换手率计算

    volume: 成交量（手），1 手 = 100 份
    float_share: 场内流通份额（万份）

    换手率 = 成交量(份) / 流通份额(份) × 100%
           = volume(手) × 100 / (float_share(万份) × 10000) × 100
    """
    if not volume or not float_share or float_share <= 0:
        return None
    float_share_in_shares = float_share * 10000  # 万份 → 份
    turnover = volume * 100 / float_share_in_shares * 100
    return round(turnover, 4)
```

### 5.3 预计收益率详解

```python
def calc_est_return(
    premium_rate: float,        # 当前溢价率 (%)
    purchase_fee_rate: float,   # 申购费率 (%)，可能为 None
    redemption_fee_rate: float, # 赎回费率 (%)，可能为 None
    commission: float = 0.0001, # 券商佣金（小数，如万1=0.0001）
    fee_discount: float = 1.0,  # 费率折扣（如0.1=一折）
) -> float | None:
    """
    预计收益率（用户参数化）

    溢价套利（申购→卖出）：
      净收益 = 卖出价 × (1 - 佣金) - 申购价 × (1 + 申购费率×折扣)
      收益率 = 净收益 / 申购价 × 100%
      简化: premium_rate/100 - purchase_fee_rate/100 × fee_discount - commission × 2

    折价套利（买入→赎回）：
      净收益 = 赎回净值 × (1 - 赎回费率) - 买入价 × (1 + 佣金)
      简化: |premium_rate|/100 - redemption_fee_rate/100 - commission × 2
    """
    if premium_rate is None:
        return None

    p_fee = (purchase_fee_rate or 0) * fee_discount / 100
    r_fee = (redemption_fee_rate or 0) / 100

    if premium_rate > 0:
        # 溢价套利
        net = premium_rate / 100 - p_fee - r_fee - commission * 2
    else:
        # 折价套利
        net = abs(premium_rate) / 100 - p_fee - r_fee - commission * 2

    return round(net * 100, 2)
```

### 5.5 类型一致性

```python
def to_float(value) -> float | None:
    """统一转 float，处理 Decimal/int/str"""
    if value is None:
        return None
    if isinstance(value, Decimal):
        return float(value)
    try:
        return float(value)
    except (ValueError, TypeError):
        return None

def to_int(value) -> int | None:
    """统一转 int"""
    f = to_float(value)
    return int(f) if f is not None else None
```

所有计算函数入口调用 `to_float`，保证 Decimal 不会混入计算。所有结果 `round(value, 4)`。

### 6.2 批量 UPSERT 实现

```python
from sqlalchemy.dialects.postgresql import insert as pg_insert

async def batch_upsert(
    session_factory,
    model,
    records: list[dict],
    conflict_columns: list[str],
    batch_size: int = 100,
) -> dict:
    """
    通用批量 UPSERT

    model: ORM 模型类
    records: 待写入数据
    conflict_columns: ON CONFLICT 的列名列表
    返回: {"total": N, "success": N, "failed_batches": [...]}
    """
    total = len(records)
    success = 0
    failed_batches = []

    for i in range(0, total, batch_size):
        batch = records[i : i + batch_size]
        batch_num = i // batch_size

        try:
            async with session_factory() as session:
                async with session.begin():
                    stmt = pg_insert(model).values(batch)
                    # ON CONFLICT DO UPDATE SET (排除主键列)
                    update_cols = {
                        c.name: stmt.excluded[c.name]
                        for c in model.__table__.columns
                        if c.name not in conflict_columns
                    }
                    stmt = stmt.on_conflict_do_update(
                        index_elements=conflict_columns,
                        set_=update_cols,
                    )
                    await session.execute(stmt)

            success += len(batch)
            logger.debug(f"batch {batch_num}: {len(batch)} rows saved")

        except Exception as e:
            failed_batches.append({
                "batch_num": batch_num,
                "start": i,
                "count": len(batch),
                "error": str(e),
            })
            logger.error(f"batch {batch_num} failed: {e}")

        # 批次间隔，给 DB 呼吸空间
        if i + batch_size < total:
            await asyncio.sleep(0.1)

    return {"total": total, "success": success, "failed_batches": failed_batches}
```

```python
async def ensure_partition(session_factory, target_date: date):
    """确保目标日期所在月的分区存在"""
    partition_name = f"asset_daily_{target_date.strftime('%Y%m')}"
    start = target_date.replace(day=1)
    end = (start + timedelta(days=32)).replace(day=1)

    async with session_factory() as session:
        async with session.begin():
            exists = await session.execute(
                text("SELECT 1 FROM pg_tables WHERE tablename = :name"),
                {"name": partition_name}
            )
            if not exists.first():
                await session.execute(text(f"""
                    CREATE TABLE IF NOT EXISTS {partition_name}
                    PARTITION OF asset_daily
                    FOR VALUES FROM ('{start}') TO ('{end}')
                """))
                logger.info(f"分区创建: {partition_name}")
```

```python
async def save_progress(session_factory, task_name: str, result: dict):
    """记录执行进度和失败信息到 fetch_progress"""
    failed_codes = [b.get("error", "") for b in result.get("failed_batches", [])]

                    total = EXCLUDED.total,

                "total": result["total"],
```

### 7.1 消费者主循环

```python
async def stream_consumer():
    """
    后台消费者：从 Redis Stream 读取事件 → 处理 → ack
    每个事件独立处理，单个失败不影响其他事件
    """
    await init_consumer_group()
    logger.info("Stream 消费者启动")

    while True:
        try:
            events = await consume_events(count=10, block_ms=2000)
            for event in events:
                try:
                    await dispatch(event)
                    await ack_event(event["id"])
                except Exception as e:
                    logger.error(f"事件处理失败: type={event.get('type')} id={event['id']} error={e}")
                    # 不 ack，下次重新消费
                    # 但要防止毒消息（一直失败的事件）：记录失败次数，超过 3 次强制 ack
                    await handle_poison_message(event, e)

        except asyncio.CancelledError:
            logger.info("Stream 消费者停止")
            break
        except Exception as e:
            logger.error(f"消费者异常: {e}")
            await asyncio.sleep(5)
```

```python
    asyncio.create_task(cross_validate_akshare(marked))
```

### 7.4 daily_save 逻辑（核心）

```python
async def process_daily_save(data: dict, batch_id: str):
    """
    日终入库：合并 Redis 中的收盘价+净值+日线 → 计算 → 写 DB
    分两步：先非 QDII，再 QDII
    """
    today = datetime.now().strftime("%Y%m%d")
    today_date = datetime.now().date()

    # 1. 从 Redis 读取数据
    closing_data = await cache_get(f"rt:close:{today}") or {}
    nav_data = await cache_get("nav:all") or {}

    # 2. 从 DB 读取基金列表和昨日数据
    async with session_factory() as session:
        code_list = await session.fetch("SELECT code, name FROM fund_code_list")
        prev_day_map = await _get_prev_day_map(session, today_date)
        recent_3d_map = await _get_recent_3d_map(session, today_date)
        qdii_codes = await _get_qdii_codes(session)

    # 3. 合并数据
    merged = []
    for fund in code_list:
        code = fund["code"]
        closing = closing_data.get(code, {})
        nav = nav_data.get(code, {})

        if not closing:
            logger.debug(f"{code} 无收盘价，跳过（停牌）")
            continue

        record = {
            "code": code,
            "name": fund.get("name") or closing.get("name"),
            "trade_date": today_date,
            "close": closing.get("realtime_price"),
            "open": closing.get("open"),
            "high": closing.get("high"),
            "low": closing.get("low"),
            "volume": closing.get("volume"),
            "amount": closing.get("realtime_amount"),
            "float_share": closing.get("float_market_cap"),  # 需确认字段
            "total_share": closing.get("total_share"),
            "limit_up": closing.get("limit_up"),
            "limit_down": closing.get("limit_down"),
            "nav": nav.get("nav"),
            "nav_date": nav.get("nav_date"),
            "nav_type": "confirmed",
            "nav_source": "lsjz",
            "fetch_source": closing.get("fetch_source", "push2"),
            "fetch_batch_id": batch_id,
        }

        # QDII 无净值处理
        if not record["nav"] and code in qdii_codes:
            prev_nav = prev_day_map.get(code, {}).get("nav")
            if prev_nav:
                record["nav"] = prev_nav
                record["nav_type"] = "estimated"
                record["nav_source"] = "estimated"

        merged.append(record)

    # 4. 计算派生字段
    calculated = []
    for fund in merged:
        result = await calc_daily_fields(
            fund,
            prev_day=fund.get("_prev_day") or prev_day_map.get(fund["code"]),
            recent_3d=recent_3d_map.get(fund["code"], []),
        )
        calculated.append(result)

    # 5. 分批写 DB
    result = await batch_upsert(session_factory, FundDaily, calculated, ["code", "trade_date"])

    # 6. 数据完整性校验
    expected_count = len(code_list)
    actual_count = result["success"]
    if expected_count > 0 and actual_count < expected_count * 0.95:
        missing = expected_count - actual_count
        await alert("数据完整性", f"daily_save: {missing}/{expected_count} 只基金缺失", "P1")

    # 7. 全部成功后刷新物化视图
    if not result["failed_batches"]:
        await refresh_materialized_view(session_factory)
    else:
        logger.warning(f"daily_save 有 {len(result['failed_batches'])} 个批次失败，不刷新物化视图")

    # 8. 记录 job_log
    await save_job_log("daily_save", result, batch_id)
```

```
    → 用昨日净值替代，nav_type = "estimated"
```

### 7.6 毒消息处理

```python
_poison_counter: dict = {}  # {event_id: fail_count}
MAX_RETRY = 3

async def handle_poison_message(event: dict, error: Exception):
    """防止毒消息（一直失败的事件）阻塞消费者"""
    event_id = event["id"]
    _poison_counter[event_id] = _poison_counter.get(event_id, 0) + 1

    if _poison_counter[event_id] >= MAX_RETRY:
        logger.error(f"毒消息丢弃: id={event_id} type={event['type']} 失败 {MAX_RETRY} 次")
        await ack_event(event_id)  # 强制 ack，不再重试
        del _poison_counter[event_id]

        # 记录到 job_log
        await save_job_log(f"poison_{event['type']}", {
            "total": 1, "success": 0,
            "failed_batches": [{"error": str(error)}]
        }, event_id)
```

```python
async def check_data_completeness(session_factory, trade_date: date, batch_id: str):

                "P1"
```
